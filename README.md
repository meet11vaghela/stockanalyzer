# Multi-Agent Stock Analysis System

A sophisticated stock analysis system using Google ADK and a team of specialized AI agents.

## Features
- **Real-time Data**: Fetches data using `yfinance`.
- **Multi-Agent Architecture**: Specialized agents for Technical, Fundamental, Sentiment, and Risk analysis.
- **Interactive Dashboard**: Plotly-based visualization.
- **Comprehensive Reports**: Generates detailed investment insights.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the system: `python main.py`

## Agents
- **Orchestrator**: Manages workflow.
- **Data Fetcher**: Retrieves market data.
- **Technical Analyst**: Analyzes charts/indicators.
- **Fundamental Analyst**: Evaluates financials.
- **Sentiment Analyst**: Processes news sentiment.
- **Risk Assessment**: Calculates risk metrics.
- **Report Generator**: Creates final reports.
